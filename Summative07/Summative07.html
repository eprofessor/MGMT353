Learning Objectives

This assignment makes you aware of what is required to setup and configure a Raspberry Pi, and how it can be used as a part of the Internet of Things (I0T).

It is a part of Project 2 and Project 4-Internet of Things

After completing this assignment you will comprehend these topics:

  1. Raspberry Pi Overview
  2. Raspberry Pi Accessories
  3. Raspberry Pi Operating Systems
  4. VNC-Virtual Network Connection
  5. GPIO-General Purpose Input/Output
  6. Hardware Add-ons
